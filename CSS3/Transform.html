Properti Transform memungkinkan kita dapat memanipulasi format visual dari elemen HTML

Ada 2 : TRANSFORMASI 2D & TRANSFORMASI 3D


transform: fungsi;

  • scale(angka): Affects the size of the element. This also applies to the font-size, padding, height, and width of an element, too. It’s also a a shorthand function for the scaleX and scaleY functions.
  • rotate(deg): Rotates the element clockwise from its current position.
  • skewX(deg) and skewY(deg): Tilts an element to the left or right, like turning a rectangle into a parallelogram. skew() is a shorthand that combines skewX() and skewY by accepting both values.
  • translate(px): Moves an element sideways or up and down.

    • EXTRA

  • matrix(): A function that is probably not intended to be written by hand, but combines all transforms into one.
  • perspective(): Doesn’t affect the element itself, but affects the transforms of descendent elements’ 3D transforms, allowing them all to have a consistent depth perspective.
