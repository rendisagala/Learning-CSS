Transition

properti transition memungkinkan kita dapat mengubah nilai dari properti HTML.

-- >durasi ( s / ms )



SENTUH UNTUK MELIHAT PERUBAHAN!!



1. durasi ( s / ms )

    :before{
        background-color: salmon;
        transition: 1s;
    }
    :hover{
        transform: rotate(180deg);
    }

2. property

    :before{
        background-color: salmon;
        transition: 1s;
    }
    :hover{
        transform: rotate(720deg);
        margin-left: 800px;
        background-color: royalblue;
    }

properti yang bisa di-transisikan

Contoh = 
    .property2:before{
        background-color: salmon;
        transition: margin-left 1s;
    }
    .property2:hover{
        transform: rotate(720deg);
        margin-left: 800px;
        background-color: crimson;
    }

ease(default)


3. fungsi

    :before{
        transition: all 1s ease;
    }
ease(default)
    :before{
        transition: all 3s ease-in;
    }
ease-in
    :before{
        transition: all 3s ease-out;
    }
ease-out
        :before{
            transition: all 3s ease-in-out;
    }
ease-in-out
    :before{
        transition: all 3s ease-in-linear;
    }
linear
    :before{
        transition: all 3s cubic-bezier(0.24, -0.35, 0.78, 1.46);
    }
cubic-bezier(w,x,y,z)

4.delay (s / ms)

tinggal menambahkan angka setelah durasi

    :before{
        transition: 3s 2s;
    }
            
    :before{
        transition: margin-left 3s, border-radius 5s, transform 3s, background-color 500ms 2s;
