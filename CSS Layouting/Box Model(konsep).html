BOX MODEL: KONSEP

Box model adalah konsep yang paling utama dari CSS Layouting, jika kalian paham mengenai ini maka akan mudah untuk kalian dalam melakukan pengaturan tata letak elemen pada website kalian.

    Ada 4 komponen pada box model
  1. margin
    2. area transparan di sekitar kotak (di luar border)
    properti margin :
    margin-top, margin-right, margin-bottom, margin-left, margin
  2. padding
    3. area transparan di dalam kotak (antara content dan border)
    properti padding :
    padding-top, padding-right, padding-bottom, padding-left, padding
  3. border
    4. batas disekeliling content dan padding
    properti border :
    border-top, border- right, border bottom, border-left, border
  4. content
    5. konten sebenarnya di dalam box, bisa berupa text atau gambar
