CSS Reset

CSS Reset adalah sebuah aktivitas css untuk menghilangkan atau mengubah default dari sebuah elemen html.
Misalnya, elemen body mempunyai margin default maka resetnya adalah membuat margin : 0

selector * untuk memilih semua elemen html [*{.......}]
